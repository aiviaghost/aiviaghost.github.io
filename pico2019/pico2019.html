asm1 - Points: 200

Problem description:

What does asm1(0x76) return? Submit the flag as a hexadecimal value (starting with '0x'). NOTE: Your submission for this question will NOT be in the normal flag format. Source located in the directory at /problems/asm1_0_b87970313ffbb5bcf4240e7c7b6c90cf.

This challenge serves an introduction to assembly. Basically you have a bunch of registers, epb, esp and eax in this case. Registers are memory locations in the CPU that can store some data and allow the CPU run computations without having to access "outside" memory locations. Each register has a special purpose but since this is a simple challenge these registers can simply be viewed as normal variables, they just store some value. The goal of this challenge will be to figure out what the final value stored in ebp is.
